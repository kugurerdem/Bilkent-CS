If Statements

If statements are used to make a decision in java. If the first condition is met, the first block of code is executed. If not, the second block is executed.

Syntax:

if(condition){
  // Code to execute if condition is met
} else {
  // Code to execute if condition is not met
}

Example:

if(i == 3){
  // Prints "i is equal to three" if i = 3
  System.out.println("i is equal to three");
} else {
  // Prints "i is not equal to three" if i != 3
  System.out.println("i is not equal to three");
}
