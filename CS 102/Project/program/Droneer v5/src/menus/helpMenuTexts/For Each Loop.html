For Each Loops

For each loops are used to repeat executing a block of code for each element of a certain list (ArrayList, Array, etc).

Syntax:

for(Object o: someList){
  // Code to execute while going through the elements of someList
}

Example:

for(Integer i: intList){
  // Prints "Hello World!" for each element of intList
  System.out.println("Hello World!");
}
